## 简易安装

Hermes中文官方文档：[https://hermes-agent.nousresearch.com/docs/zh-Hans/](https://hermes-agent.nousresearch.com/docs/zh-Hans/)

![](http://rainyudianxx.baimuxym.cn/hellocoder/image-20260611013048475.png)





wsl 安装

```bash
curl -fsSL https://hermes-agent.nousresearch.com/install.sh | bash
```

window安装（**PowerShell**）

```
iex (irm https://hermes-agent.nousresearch.com/install.ps1)
```



简简单单一句命令即可。

## Hermes和openclaw有什么区别

作为深度体验过这两个智能体框架的用户，`openclaw`横空出世的时候，热度非常高，但是！它的门槛真的非常高，普通人安装完毕了，就没有下文了，顶多就接入聊天工具像飞书什么的，然后他聊聊天。



而且这个 openclaw 的问题也特别多，我在使用的时候，有些错误异常的诡异，用起来，各种水土不服，那时候我就觉得这个东西一定会被替代的。（有一种雷声大雨点小的感觉）



后面腾讯出了 Qclaw、workbuddy ，我觉得这两个比 openclaw 更好用，普通人更适合（懒得折腾，就是免费额度有点少）



Hermes 爆火的时候，我以为个东西也和 openclaw一样，所以我是持观望态度的，毕竟这么多AI技术，一天一个样，跟本学不完。



不过，当我接触Hermes后，这个东西对新手来说还是很友好的，相对来说，还是挺有帮助的。



**在“长期陪伴、自我进化”这个维度上，Hermes的优点确实更突出，甚至可以说是它的核心优势。**
