我们重视您的隐私。本隐私政策说明当您访问、浏览本网站时，我们可能如何收集、使用、存储及保护相关信息。请在使用本网站前仔细阅读本政策。

## 1. 我们收集的信息

本网站以公开阅读技术文章为主，**通常不要求您注册账号或提交个人信息**。但在以下场景中，可能会产生少量数据：

### 1.1 您主动提供的信息

- 通过邮件（[huangyongwen0306@163.com](mailto:huangyongwen0306@163.com)）联系我们时，您提供的邮箱地址、邮件内容；
- 在已开启评论功能的页面发表评论时，您通过第三方评论服务提交的内容（如 GitHub 账号相关信息）；
- 访问部分设置了访问密码的文章时，您输入的密码仅用于本地校验，不会被上传至我们的服务器。

### 1.2 自动收集的信息

当您访问本网站时，服务器及第三方分析/广告服务可能会自动记录：

- IP 地址、浏览器类型与版本、操作系统、设备类型；
- 访问时间、访问页面 URL、来源页面（Referrer）；
- Cookie、本地存储（Local Storage）中的偏好设置（如主题模式）；
- 页面浏览、点击、停留时长等统计信息。

上述信息通常无法单独识别您的真实身份，但可能与其他信息结合后用于分析访问行为。

## 2. 我们如何使用这些信息

我们可能将收集到的信息用于：

- 提供、维护和改进本网站的内容与访问体验；
- 统计访问量、分析热门内容、了解读者偏好；
- 展示个性化或上下文相关的广告（如已启用）；
- 排查故障、保障网站安全、防止滥用；
- 回复您通过邮件等方式提出的咨询。

**我们不会出售您的个人信息。**

## 3. Cookie 与本地存储

本网站及集成的第三方服务可能使用 Cookie 或浏览器本地存储，用于：

- 记住您的主题偏好（如浅色/深色模式）；
- 统计访问数据；
- 广告投放与效果衡量；
- 部分文章访问控制相关的本地校验。

您可以通过浏览器设置拒绝或删除 Cookie。但关闭 Cookie 可能导致部分功能（如主题记忆、统计、广告）无法正常工作。

为实现统计、广告、评论等功能，本网站可能接入以下第三方服务。这些服务由各自运营方独立提供，其数据处理行为受对应隐私政策约束：

- 统计数据主要由第三方分析平台处理与存储；
- 我们采取合理措施保护网站安全，但**无法保证互联网传输或存储的绝对安全**。
