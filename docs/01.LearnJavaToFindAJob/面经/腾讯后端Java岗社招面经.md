###  三面： 

 1、go程和线程有什么区别？ 

 答：起一个go程大概只需要4kb的内存，起一个Java线程需要1.5MB的内存；go程的调度在用户态非常轻量，Java线程的切换成本比较高。 

 2、接着问为什么成本比较高？ 

 答：因为Java线程的调度需要在用户态和内核态切换所以成本高
