## 二、免费版自带的核心基础功能

免费版仍保留的核心功能

尽管有上述阉割，但免费版仍保留了数据库管理的核心能力，足以应对日常开发任务：

- **多数据库连接**：支持同时连接 **MySQL、PostgreSQL、SQL Server、Oracle、MariaDB、SQLite** 等SQL数据库，以及 **Redis、MongoDB** 等NoSQL数据库。
- **数据查看与编辑**：提供网格视图、树视图和JSON视图来查看和编辑数据。
- **查询编辑器**：具备代码补全、代码片段和语法高亮功能，方便编写SQL。
- **导入与导出**：支持从TXT、CSV、XML、JSON等格式文件导入和导出数据。
- **对象设计器**：提供清晰、响应式的界面来管理数据库表、视图等对象结构。
- **协同合作**：支持将连接设置、查询等同步到云端，便于团队协作。

##  三、功能削减

与功能全面的收费完整版相比，免费版在以下方面做了明显的功能削减或限制：

| 功能模块           | 阉割/限制情况 | 说明                                                         |
| :----------------- | :------------ | :----------------------------------------------------------- |
| **数据同步与传输** | **完全移除**  | 不支持完整版中的数据同步、数据传输、结构同步等功能。         |
| **数据建模与BI**   | **完全移除**  | 不支持模型功能和BI（商业智能）图表功能，完整版中超过20种的图表类型在免费版中不可用。 |
| **备份与还原**     | **功能削减**  | 仅支持基础的转储（导出）或运行（导入）SQL文件，缺少完整版的高级备份恢复选项。 |
| **自动化**         | **完全移除**  | 不支持自动化运行任务等功能。                                 |
| **SQL美化**        | **曾有限制**  | 早期免费版不支持SQL格式化功能，但后续的更新（版本17.1.2及以上）已加入此功能。 |
| **高级分析**       | **功能削减**  | 在数据分析和报表生成等高级功能上有所缺失                     |
